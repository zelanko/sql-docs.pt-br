|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|Inicia no registro atual.|  
|**adBookmarkFirst**|1|Inicia no primeiro registro.|  
|**adBookmarkLast**|2|Inicia no último registro.|
