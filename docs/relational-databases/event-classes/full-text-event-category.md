# <a name="full-text-event-category"></a>Categoria de evento Full Text
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  A categoria de evento **Full Text** contém eventos gerais de texto completo.  
  
## <a name="in-this-section"></a>Nesta seção  
  
|Tópico|Descrição|  
|-----------|-----------------|  
|[Classe de evento FT:Crawl Aborted](../../relational-databases/event-classes/ft-crawl-aborted-event-class.md)|Indica que um rastreamento de texto completo encontrou uma exceção.|  
|[Classe de evento FT:Crawl Started](../../relational-databases/event-classes/ft-crawl-started-event-class.md)|Indica que um rastreamento de texto completo foi iniciado.|  
|[Classe de evento FT:Crawl Stopped](../../relational-databases/event-classes/ft-crawl-stopped-event-class.md)|Indica que um rastreamento de texto completo foi interrompido.|
