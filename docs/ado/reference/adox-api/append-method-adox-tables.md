# <a name="append-method-adox-tables"></a>Método Append (Tabelas do ADOX)
Adiciona um novo [tabela](../../../ado/reference/adox-api/table-object-adox.md) do objeto para o [tabelas](../../../ado/reference/adox-api/tables-collection-adox.md) coleção.  
  
## <a name="syntax"></a>Sintaxe  
  
```  
  
Tables.Append Table  
```  
  
#### <a name="parameters"></a>Parâmetros  
 *Table*  
 Um **Variant** valor que contém uma referência para o **tabela** para acrescentar ou o nome da tabela para criar e anexar.  
  
## <a name="remarks"></a>Comentários  
 Se o provedor não dá suporte a criação de tabelas, ocorrerá um erro.
