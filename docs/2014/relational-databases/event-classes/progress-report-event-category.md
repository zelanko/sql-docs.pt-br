  A categoria de evento **Progress Report** contém a classe de evento **Progress Report: Online Index Operation** .  
  
|Tópico|Description|  
|-----------|-----------------|  
|[Classe de evento Progress Report: Online Index Operation](progress-report-online-index-operation-event-class.md)|Indica o progresso de uma operação de compilação de índices online.|
