# <a name="visual-studio-analyzer"></a>Analisador do Visual Studio
> [!IMPORTANT]  
>  Suporte para o analisador do Visual Studio foi removido a partir do Windows 8 (Visual Studio Analyzer foi incluído apenas em versões mais antigas do Visual Studio.). Para obter uma alternativa de mecanismo de solução de problemas, use o rastreamento da oferta.  
  
 Microsoft® Visual Studio™ Analyzer fornece um modo de exibição de alto nível do aplicativo ODBC. O desenvolvedor pode avaliar, analisar e depurar a estrutura, desempenho e interações de perspectiva do aplicativo, em vez de uma perspectiva de componente ou o código. As informações coletadas pelo analisador do Visual Studio estão na forma de eventos, que representam algum tipo de interação entre dois componentes do aplicativo. Eventos podem ser compostos de chamadas de função e retorna das funções, como conexões, consultas de banco de dados e transações.
