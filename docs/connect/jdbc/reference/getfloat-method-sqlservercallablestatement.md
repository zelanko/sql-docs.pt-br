# <a name="getfloat-method-sqlservercallablestatement"></a>Método getFloat (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera o valor do parâmetro designado como um **float** na linguagem de programação Java.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Descrição|  
|----------|-----------------|  
|[getFloat (int)](../../../connect/jdbc/reference/getfloat-method-int.md)|Recupera o valor do parâmetro designado como um **float** na linguagem de programação Java, considerando o índice do parâmetro.|  
|[getFloat (java.lang.String)](../../../connect/jdbc/reference/getfloat-method-java-lang-string.md)|Recupera o valor do parâmetro designado como um **float** na linguagem de programação Java, considerando o nome do parâmetro.|
