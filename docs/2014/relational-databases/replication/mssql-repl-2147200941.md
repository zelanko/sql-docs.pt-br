# <a name="mssqlrepl-2147200941"></a>MSSQL_REPL-2147200941
    
## <a name="message-details"></a>Detalhes da mensagem  
  
|||  
|-|-|  
|Nome do produto|SQL Server|  
|ID do evento|-2147200941|  
|Origem do evento|MSSQLServer|  
|Nome simbólico||  
|Texto da mensagem|O processo de mesclagem detectou uma incompatibilidade ao avaliar a expressão de validação da partição do assinante. O problema pode ser resolvido reinicializando-se a assinatura.|

## <a name="user-action"></a>Ação do usuário  
 Reinicialize a assinatura.
