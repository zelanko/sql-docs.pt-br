# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>Método setMultiSubnetFailover (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Define o valor da propriedade de conexão **multiSubnetFailover**.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>parâmetros  
 *multiSubnetFailover*  
  
 O novo valor da propriedade de conexão **multiSubnetFailover**.  
  
## <a name="remarks"></a>Comentários  
 Confira mais informações sobre a propriedade de conexão **multiSubnetFailover**, confira [Configuração das propriedades de conexão](../../../connect/jdbc/setting-the-connection-properties.md).
