# <a name="configurationsetting-property---virtualdirectoryreportmanager"></a>Propriedade de ConfigurationSetting – VirtualDirectoryReportManager
  Retorna o diretório virtual para o aplicativo do gerenciador de relatórios. Somente leitura.  
  
## <a name="syntax"></a>Sintaxe  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## <a name="property-values"></a>Valores da propriedade  
 Uma **cadeia de caracteres** que contém o diretório virtual.
