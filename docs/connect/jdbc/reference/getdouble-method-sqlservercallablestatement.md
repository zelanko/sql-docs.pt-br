# <a name="getdouble-method-sqlservercallablestatement"></a>Método getDouble (SQLServerCallableStatement)

|Nome|Descrição|  
|----------|-----------------|  
|[getDouble (int)](../../../connect/jdbc/reference/getdouble-method-int.md)|Recupera o valor do parâmetro designado como um **double** na linguagem de programação Java, considerando o índice do parâmetro.|  
|[getDouble (java.lang.String)](../../../connect/jdbc/reference/getdouble-method-java-lang-string.md)|Recupera o valor do parâmetro designado como um **double** na linguagem de programação Java, considerando o nome do parâmetro.|
