# <a name="odbc-sql-grammar"></a>Gramática SQL ODBC
As aplicações interoperáveis devem sempre usar a gramática SQL ODBC em instruções SQL. No entanto, uma variação considerável é possível mesmo dentro desta gramática. Para obter mais informações, consulte [Interoperabilidade das Instruções SQL](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
