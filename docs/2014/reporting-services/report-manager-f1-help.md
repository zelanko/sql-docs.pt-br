# <a name="report-manager-f1-help"></a>Ajuda F1 do Gerenciador de Relatórios
  Os tópicos a seguir fornecem ajuda em nível de [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../includes/ssrsnoversion-md.md)] página para Report Manager. Há um tópico para cada página no aplicativo. Nem todas as páginas podem estar disponíveis. As configurações de segurança determinam as páginas que você pode acessar. Para acessar esses tópicos do Gerenciador de Relatórios, clique em **Ajuda** na barra de ferramentas principal.
