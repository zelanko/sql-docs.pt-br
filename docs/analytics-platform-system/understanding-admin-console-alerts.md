# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>Noções básicas sobre alertas do console de administração no Analytics Platform System
Os alertas são exibidos no **console de administração** do dispositivo e no **System Center Operations Manager** (SCOM). Use esta lista de alertas para ajudar a identificar quais alertas exigem investigação adicional.  
  
Para obter informações sobre como se conectar ao **console de administração** usando o Internet Explorer, consulte [monitorar o dispositivo usando o console de administração &#40;&#41;do sistema de plataforma de análise ](monitor-the-appliance-by-using-the-admin-console.md). Para obter informações sobre o **SCOM**, consulte [monitorar o dispositivo usando System Center Operations Manager &#40;o sistema de plataforma de análise&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
Para obter informações sobre como obter informações de alerta\-usando o Transact SQL, consulte [monitorar o dispositivo usando as exibições do sistema &#40;&#41;do sistema de plataforma de análise ](monitor-the-appliance-by-using-system-views.md).  
  
## <a name="types-of-alerts"></a>Tipos de alertas  
Os nomes de alerta que indicam um status NORMAL geralmente não exigem investigação. Os nomes de alerta que contêm as palavras NON_CRITICAL às vezes exigem uma ação. A investigação é necessária para todos os outros tipos de alertas.
