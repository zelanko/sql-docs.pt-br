# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  A interface IBCPSession2 é uma extensão IBCPSession que fornece uma função de membro que é uma alternativa à chamada ibcpsession:: BCPColFmt para cada coluna.  IBCPSession2 herda IBCPSession e adiciona um novo método: [IBCPSession2::BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
