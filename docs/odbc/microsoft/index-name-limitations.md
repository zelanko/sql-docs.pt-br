# <a name="index-name-limitations"></a>Limitações do nome do índice
Quando o driver do Paradox é usado, um índice primário deve ter o mesmo nome que a tabela na qual ele está definido. Outros índices exclusivos ou não exclusivos devem ter o mesmo nome que a tabela na qual eles são definidos.
