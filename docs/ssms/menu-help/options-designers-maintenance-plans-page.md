# <a name="options-designers---maintenance-plans-page"></a>Opções (página Designer – Planos de manutenção)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Use a caixa de diálogo **Opções** para determinar o comportamento padrão do designer quando novas formas forem adicionadas à superfície de design. No menu **Ferramentas** , clique em **Opções**, expanda **Designers**e selecione **Planos de Manutenção**.  
  
## <a name="uielement-list"></a>Lista de elementos de interface do usuário  
**Conecte uma nova forma à forma selecionada por padrão**  
Quando essa caixa de seleção está selecionada, novas formas conectam-se automaticamente a outras formas ao serem adicionadas à superfície de design. Quando essa caixa de seleção está desmarcada, novas formas não são conectadas ao serem adicionadas. Selecionar essa caixa de seleção habilita as seguintes caixas.  
  
**Especifique a conexão da nova forma**  
Selecione nessa caixa de listagem suspensa para designar o padrão para o tipo de conexão estabelecida entre as formas. As três opções são:  
  
-   **Use uma restrição de Êxito para a nova forma**  
  
-   **Use uma restrição de Falha para a nova forma**  
  
-   **Use uma restrição de Conclusão para a nova forma**  
  
**Especifique o local da nova forma**  
Selecione nessa caixa de listagem suspensa para designar o local padrão onde uma nova forma é adicionada à superfície de design. As quatro opções são:  
  
-   **Adicione a nova forma à esquerda da forma selecionada**  
  
-   **Adicione a nova forma à direita da forma selecionada**  
  
-   **Adicione a nova forma acima da forma selecionada**  
  
-   **Adicione a nova forma abaixo da forma selecionada**
