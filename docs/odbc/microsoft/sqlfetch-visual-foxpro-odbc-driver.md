# <a name="sqlfetch-visual-foxpro-odbc-driver"></a>SQLFetch (Driver ODBC do Visual FoxPro)
> [!NOTE]  
>  Este tópico contém informações específicas de Driver ODBC do Visual FoxPro. Para obter informações gerais sobre essa função, consulte o tópico apropriado sob [referência da API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Suporte: completo  
  
 Conformidade com a API ODBC: Nível de núcleo  
  
 Recupera uma linha de um conjunto de resultados em locais especificados pelas chamadas anteriores ao [SQLBindCol](../../odbc/microsoft/sqlbindcol-visual-foxpro-odbc-driver.md). Prepara o driver para uma chamada para [SQLGetData](../../odbc/microsoft/sqlgetdata-visual-foxpro-odbc-driver.md) para as colunas não associadas.  
  
 Para obter mais informações, consulte [SQLFetch](../../odbc/reference/syntax/sqlfetch-function.md) na *referência do programador de ODBC*.
