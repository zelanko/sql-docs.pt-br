# <a name="deprecated-integration-services-features-in-sql-server-2014"></a>Recursos dos serviços de integração preteridos no SQL Server 2014
  Este tópico descreve os [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] recursos preteridos que ainda estão disponíveis na versão atual do [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)]. Esses recursos estão programados para serem removidos em uma versão futura do [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]. Recursos preteridos não devem ser usados em aplicativos novos.  
  
 Não há nenhum recurso preterido do [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] no [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)].
