# <a name="run-queries-visual-database-tools"></a>Executar consultas (Visual Database Tools)
Quando você tiver terminado de criar a consulta, execute-a.  
  
### <a name="to-execute-a-query"></a>Para executar uma consulta  
  
1.  Abra ou crie a consulta que deseja executar.  
  
2.  Clique com o botão direito do mouse na janela de consultas e selecione **Executar SQL** no menu de atalho.  
  
    –ou–  
  
    Pressione CTRL+R.  
  
Se você estiver criando uma consulta Select, os resultados da consulta aparecerão no painel Resultados.  
  
O Designer de Consulta e Exibição retorna resultados para o seu computador em lotes (em incrementos), de modo que você pode começar a exibir os resultados rapidamente. Dessa maneira, é possível executar outras tarefas enquanto a consulta está em andamento.  
  
Quando você criar uma consulta Update, Insert From, Insert Into, Delete ou Make Table, o Designer de Consulta e Exibição exibirá uma mensagem mostrando o número de linhas afetadas pela consulta.
