# <a name="execute-method-sqlserverstatement"></a>Método execute (SQLServerStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Executa a instrução SQL fornecida que pode retornar vários resultados.  
  
## <a name="overload-list"></a>Lista de sobrecargas  
  
|Nome|Description|  
|----------|-----------------|  
|[Execute (Java)](../../../connect/jdbc/reference/execute-method-java-lang-string-sqlserverstatement.md)|Executa a instrução SQL fornecida que pode retornar vários resultados.|  
|[Execute (Java, int)](../../../connect/jdbc/reference/execute-method-java-lang-string-int.md)|Executa a instrução SQL fornecida, que pode retornar vários resultados e sinaliza para [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)] que todas as chaves geradas automaticamente devem ser disponibilizadas para recuperação.|  
|[Execute (Java, int&#91;&#93;)](../../../connect/jdbc/reference/execute-method-java-lang-string.md)|Executa a instrução SQL fornecida, que pode retornar diversos resultados, e sinaliza ao driver JDBC que as chaves geradas automaticamente indicadas na matriz fornecida devem ser disponibilizadas para recuperação.|  
|[Execute (Java, Java&#91;&#93;)](../../../connect/jdbc/reference/execute-method-java-lang-string-java-lang-string.md)|Executa a instrução SQL fornecida, que pode retornar diversos resultados, e sinaliza ao driver JDBC que as chaves geradas automaticamente indicadas na matriz fornecida devem ser disponibilizadas para recuperação.|
